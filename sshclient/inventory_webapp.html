SAN SSH Inventory - Password Manager + Connector

Minimal static UI. Uses backend endpoints in api/app.py:
- POST /api/credentials/save to save encrypted credentials into MongoDB
- POST /api/ssh-ring/exec to connect + run a command
Set API_BASE below if you open this HTML via file://.

Add / Save SSH Credentials

Credentials are stored server-side encrypted. Password is never stored in browser on success.
If you refresh without backend, the list may be empty because this minimal app doesn’t implement a secure “list credentials” API.

Run commands against saved credentials

Since there is no API to list decrypted credentials, this minimal UI keeps a small local cache of (ip, username, port, device_name).
The password is sent only when you press “Run commands”.
