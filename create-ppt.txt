Project Details
Hardware Test Ring Management Tool:
To develop Tool/Infrastructure Analytics Engine to view all equipment connectivity details, 
along with components health. 
•
Objective/Abstract:
To solve the complex & challenging management (Refer next slide for sample Hardware 
Test Ring) of HPE Alletra Storage MP B10000 System Test Hardware rings. Ensuring all 
equipment's and components health state, Easy maintaining of connectivity details.
  It should also provide Infrastructure Analytics Engine to query supported Hardware 
details such as FC, iSCSI, NVMeoTCP and test cases that are applicable based on 
reference/input details.

Project – Implementation Breakdown Details
   Implementation Breakdown details of this project:
a) Extract key information from the provided Arrays, Hosts, Switches based on 
sample/standard queries. Run through set of commands to above devices with 
periodic/need bases intervals and collect the required details & store it in DB.

also support future addition/removal of fields/parameters. Additionally, Every DB details 
provides export option into spreadsheet.
c)   Create framework that translates English statements into DB query.
d)   Show up the stored details/queries result on Web based User Interface.

HPE CPP-3 PROJECT
Topology Intelligence & 
Health Chatbot for Storage 
Test Ring Environments
Infrastructure Analytics Engine for SAN Topology Management
Team: Preetham M R · Sachetan I · Samarth K N · Unmesh Raj  
Mentor: Dr. Mohana
The Problem: Managing Complexity Without Intelligence
Enterprise storage test rings span hundreds of 
interconnected components — hosts, switches, 
HBAs, controllers, and arrays — yet infrastructure 
knowledge remains fragmented across 
spreadsheets, disconnected diagrams, and tribal 
expertise. • No centralized topology intelligence or 
single source of truth
• Connectivity tracing is manual, slow, and 
error-prone
• Health visibility is reactive rather than 
proactive
• Unstructured logs and multiple data 
sources impede rapid troubleshooting
• Zoning relationships are opaque and 
undocumented
SOLUTION OVERVIEW
Proposed Solution: Topology Intelligence & Health 
Chatbot
A graph-based infrastructure intelligence platform that transforms raw SAN telemetry into a queryable, visualized, and health-aware topology model — 
eliminating manual tracking and enabling deterministic, intent-driven operations.
Graph-Based Topology
Neo4j infrastructure model with nodes, edges, and properties
Automated Discovery
SNMP, SSH CLI parsing, and REST API ingestion
Health Monitoring
Real-time status, degradation detection, and alerting
Intent-Based Chatbot
GraphRAG-powered natural language topology queries
Digital Twin
Simulation-based environment mirroring live test ring behavior
Visualization Dashboard
Interactive topology explorer with radial quick-query interface
ARCHITECTURE
System Architecture
Data flows deterministically from physical devices through collectors and normalizers into the Neo4j graph, where the query and health engines power both 
the chatbot interface and the interactive dashboard — with no live hardware dependency.
Graph Database & Digital Twin
Infrastructure Graph Model
All SAN components are modeled as typed nodes 
and labeled edges, enabling deterministic 
Cypher traversal across the full topology.
• Nodes: Host, Switch, Array, Controller, Port, 
Disk
• Edges: CONNECTED_TO, ZONED_WITH, 
HAS_PORT, HOSTS
• Layers: Physical, SAN, Protocol, Location
Digital Twin simulation mirrors live test 
ring behavior using synthetic telemetry 
— enabling safe topology exploration 
without hardware access.
CORE TECHNOLOGY
QUERY ENGINE
Intent-Based Query & Analytics
Engine Example Enterprise Queries

zoned with Host H1?"
"Show path from Host 
H2 to Array A4"
"Which switches are

How It Works
Natural language intents are parsed via GraphRAG, translated to 
deterministic Cypher queries, and executed against the Neo4j graph — 
delivering precise, reproducible results without generic LLM hallucination.
• Intent classification and entity extraction
• Cypher query generation with schema validation
• Graph traversal with path resolution
• Health analytics layered on topology results
USER INTERFACE
Chatbot & Dashboard UI

 inspection
 Query Assistant
Intent-based chatbot accepting natural language infrastructure queries
 Health 
Indicators Real-time status badges, degradation alerts, and port-level health
 scores
Radial Quick-Query
Contextual radial menu for rapid topology insights and common queries
